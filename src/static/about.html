About

This application was built with Express.js and TypeScript. The API routes follow a RESTful pattern, and the "/" and "/about" pages are built with vanilla JS, HTML, and SCSS. The API connects to a MongoDB database to store data.

Technologies used

This app uses bcrypt for encryption of passwords, mongoose for database management, cors for cross-origin resource handling, jsonwebtoken for producing authentication tokens, lodash for utilities, morgan for logging, zod for validation and jest and supertest for testing.
