Welcome to the back-end for UPC Tracker

This is the back-end for my UPC tracking project. The app is intended to allow me to easily keep lists of UPCs and share them between my phone, a web app, and other applications that I use at work. The records are stored on a MongoDB database and can be copied from the front end to excel or other applications.

I have already completed a working React Native front-end which is available as both a web app and an android app available on the play store. I plan to add a Vue.js and AngularJS front end as well.
